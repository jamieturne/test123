Сценарии для демо

Как это работает для клиента

Клиент пишет запрос — Spotlight понимает контекст, вызывает нужный навык и возвращает ответ

Тарифы и остатки
Клиент спрашивает
«Какой у меня тариф и сколько на счёте?»
AI определяет навык → вызывает API тарифов и счетов → отвечает: тариф «Быстрые платежи», остаток 1 247 500 ₽, дата следующего списания.
⚡ Без Spotlight: найти раздел «Тарифы» → открыть счёт → посмотреть детали
Перевод по СБП
Клиент спрашивает
«Переведи 50 000 по СБП на +7 900 123-45-67»
AI парсит сумму и номер → подготавливает перевод → клиент нажимает «Подтвердить» прямо в чате.
⚡ Без Spotlight: «Платежи» → «Новый» → «СБП» → заполнить форму → подтвердить
Выставление счетов
Клиент спрашивает
«Выстави счёт ООО "Рога и Копыта" на 150 000 ₽ за консалтинг»
AI находит контрагента → заполняет реквизиты → формирует счёт → счёт готов, подпишите.
⚡ Без Spotlight: «Счета» → «Создать» → выбрать контрагента → заполнить 7 полей
Блокировка счёта
Клиент спрашивает
«Мой счёт заблокирован, что делать?»
AI проверяет статус → определяет причину → выдаёт пошаговую инструкцию по разблокировке с перечнем документов.
⚡ Без Spotlight: звонок в поддержку → ожидание → переключение на специалиста

Roadmap подключения

11 шагов от идеи до продакшена. Простой сценарий — 1 спринт, сложный — 2-3.

0
Приоритизация
1–3 дня
Ответственный: PO / Stream Lead
Выберите 1–3 сценария (например: «покажи операции», «какой тариф»), проверьте наличие API, оцените сложность, выделите 1 разработчика на 1–2 спринта
1
Подготовка карточки навыка
1–2 дня
Ответственный: PO + Системный аналитик
Заполните карточку навыка (название, команда, контакты, сценарии, API-ручки), отправьте на spotlight-platform@alfaintra.net, дождитесь согласования
2
Получение доступов
1–2 дня
Ответственный: Разработчик
Репозиторий в MCPHUB, доступ к AI Flow, тестовый client_id в Keycloak, согласование ролевой модели с ДКБ
3
Разработка MCP-сервера
1–2 спринта
Ответственный: Разработчик
Инициализация проекта (Java/Python), реализация инструментов (@Tool), проброс JWT, логирование, локальное тестирование через MCP Inspector
4
Публикация
2–3 дня
Ответственный: Разработчик
Code review от платформы, AppSec-апрув, заказ сетевых доступов, деплой в test/prod через Jira (MCPHUB_DEPLOYMENT)
5
Подключение к AI Flow
1 день
Ответственный: Разработчик
Создание сценария в AI Flow, настройка MCP Tool, маппинг параметров, тестирование через встроенный чат
6
Подключение к Spotlight
1–2 дня
Ответственный: Разработчик + Команда Spotlight
Регистрация навыка в маршрутизаторе Spotlight, настройка триггеров и интентов, проверка корректности вызова навыка из UI Spotlight
7
E2E-тестирование
2–3 дня
Ответственный: Разработчик + Команда Spotlight + PO
Совместное тестирование в реальном UI Spotlight, проверка всех сценариев, устранение дефектов, поэтапная раскатка на пользователей
8
Планирование релиза
1–2 дня
Ответственный: PO + Команда Spotlight
Согласование даты релиза, подготовка release notes, определение аудитории раскатки (feature flags / % пользователей), план отката при инцидентах
9
Обновление контента
По необходимости
Ответственный: Разработчик + PO
Обновление флоу в AI Flow, доработка сценариев и промптов, обновление MCP-сервера (новые @Tool, изменение логики), повторное тестирование изменённых компонентов
10
Пост-релизный мониторинг
Постоянно
Ответственный: PO + Разработчик
Мониторинг метрик (обращения, конверсия, снижение нагрузки на саппорт), сбор обратной связи, планирование расширения функционала

Мотивация и бенефиты

Что получает ваша команда и ваши клиенты, когда продукт становится навыком Spotlight

Новый AI-канал бесплатно
Все пользователи Альфа-Бизнес получают доступ к вашему продукту через диалог с AI. Spotlight уже встроен в интернет-банк.
Меньше нагрузка на саппорт
Типовые вопросы — статусы, остатки, тарифы — решает AI. Минус 20% обращений в поддержку без потери качества.
Конверсия ×3
AI подводит клиента к действию в контексте разговора. Конверсия в целевое действие в 3 раза выше, чем через навигацию.
Экстра-сценарии
Сценарии, невозможные в обычном UI: «Почему мой счёт заблокирован?», «Выстави счёт контрагенту», «Какой тариф выгоднее?»
1 спринт, 1 разработчик
Не нужно переписывать API. Оборачиваете готовые ручки в MCP-сервер по шаблону. Платформа берёт на себя LLM, роутинг и UI.
Удержание клиентов
Клиенту нужно меньше действий — он не уходит, не найдя функцию. Retention D7 выше на 15% у пользователей Spotlight.

Готовы подключиться?

Передайте инструкцию разработчику — всё остальное он сделает по шаблону. Мы поможем на каждом шаге.
